
**Typography:** Import Inter from Google Fonts. Use Inter everywhere. Reference numbers, status codes, and counts use `font-variant-numeric: tabular-nums` and `font-family: 'Inter Mono'`.

**Spacing:** Use only these values: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64px. No arbitrary values.

**Design direction:** Calm authority. Minimalist. Clean whitespace. Inspired by the Antigravity homepage — bold but simple typography, clear hierarchy, zero clutter. Every element earns its place or gets removed.

## Component Rules

**Navbar (desktop):** White background, 1px bottom border `--border`. Left: `Logo.tsx`. Center: nav links in `--text-secondary`, active link `--text-primary` with 2px `--accent-primary` underline. Right: notification bell with unread count badge.

**Bottom Nav (mobile):** White, 1px top border. Icons: Home, PulseLine, RapidDispatch, VoiceBox, More (sheet for rest). Active icon `--accent-primary`, inactive `--text-muted`.

**Buttons:** Primary = `--accent-primary` bg, white text, `--radius-btn`, 40px height, sentence case. Secondary = white bg, `--border` border. Destructive = `--error` bg. All buttons show spinner on loading state.

**Inputs:** White bg, `--border` border, `--radius-btn`, 40px height. Focus = 2px `--accent-secondary` outline. Error state = `--error` border with message below.

**Status badges:** Pill shape, Inter Mono, 12px. Submitted = `#FEF3C7` bg `#D97706` text. Under Review = `#DBEAFE` bg `#1B4FD8` text. Resolved = `#DCFCE7` bg `#16A34A` text. Rejected = `#FEE2E2` bg `#DC2626` text.

**Cards:** White bg, `--border` border, `--radius-card`, `--shadow-card`. Hover = `translateY(-1px)` only, no color change.

**Toasts:** Bottom-right desktop, bottom-center mobile. 320px wide, `--radius-card`. Success = 4px left border `--success`. Error = 4px left border `--error`. Auto-dismiss 4 seconds.

**Tables (admin):** No zebra striping. `--border` row dividers. Paginated 10 rows per page. Row hover = `--bg-base` tint.

**Modals:** Centered, max 480px wide, white bg, `--radius-card`, backdrop `rgba(0,0,0,0.4)`. Close X top-right.

## Pages

### User Pages (No Login Required)

**`/` Home:** Logo.tsx centered, tagline, SOS pulse ring button linking to `/rapiddispatch`, 4-column feature card grid (desktop) / 2-column (mobile), Notice Board from `/api/notices` pinned only, Notifications widget showing 3 latest unread.

**`/pulseline`:** Sticky search bar, horizontal category filter pills, contacts from `/api/contacts`, list cards with Copy + Call Now buttons, Shadcn Dialog on click, Favourites via localStorage.

**`/rapiddispatch`:** Four 2x2 action cards with accordion forms. Report Accident (Location, Type, People, Description, Photo), Request Vehicle (Location, Vehicle Type, Urgency, Contact), Report Natural Hazard (Hazard Type, Location, Severity, Description, Photo), Report Other (Title, Location, Description, Photo). On submit: POST to `/api/reports`, POST to `/api/notifications`, show reference number RQ-YYYYMMDD-XXXX in Inter Mono. My Reports lookup by reference number below.

**`/voicebox`:** Three Shadcn Tabs. Complaint Box: form POST `/api/complaints`, show complaint ID. Discussion Board: post form POST `/api/posts`, feed with upvotes + replies + category filter. My Submissions: complaint ID lookup.

**`/aiddrop`:** Five Shadcn Tabs. Blood: form + searchable registry. Organs: form + registry. Food & Clothes: listing form + public board. Money: campaigns with progress bars + donation form. My Listings: session list.

**`/safelink`:** Two Shadcn Tabs. Info Links: from `/api/links`, search + filter + Visit buttons. Documents: from `/api/documents`, search + filter + Download buttons to `/public/docs/`.

**`/safeshield`:** `navigator.onLine` detection. Offline = full red banner, content disabled. Online = companies from `/api/insurance` with Shadcn Accordion plans + Get Insured affiliate buttons. Disclaimer at bottom.

**`/charity`:** Active campaigns from `/api/charity`, progress bars, Top Donors list, donation form per campaign, Past Campaigns collapsible section at bottom.

### Admin Pages (Login Required)

Protect all `/admin/*` routes — redirect to `/admin` if no valid localStorage session.

**`/admin`:** Login form, validate against `admin_session`, set localStorage token, redirect to dashboard on success.

**`/admin/dashboard`:** Stats cards (Total Reports, Open Complaints, Active Donors, Charity Fund, Insurance Listings), Recent Activity Feed (latest 10 across all tables), Quick Actions (Add Contact, Post Notice, Upload Document), Notifications panel, Logout button.

**`/admin/pulseline`:** Paginated table of contacts. Add / Edit / Delete with form fields: Name, Number, Category, Description.

**`/admin/rapiddispatch`:** Paginated table of reports filtered by type/date/status. Update status + add internal notes.

**`/admin/voicebox`:** Complaints table with status update + admin response. Discussion posts with hide/pin/delete moderation.

**`/admin/aiddrop`:** Four tabs. Blood Donors / Organ Donors / Food & Clothes — paginated tables with Approve/Reject/Delete. Money Campaigns — from `money_campaigns` with Create/Close/View Donors modal.

**`/admin/safelink`:** Two tabs. Info Links CRUD (Name, URL, Description, Category). Documents — upload to `/public/docs/`, insert record with file size.

**`/admin/safeshield`:** Table of insurance companies. Add/Edit/Delete/Toggle Active. Form: Name, Description, Plans JSON, Affiliate Link, Logo Upload to `/public/images/`.

**`/admin/charity`:** Table of charity campaigns. Create/Edit/Close/Archive. View Donors modal per campaign. Progress bar per row.

**`/admin/notices`:** Table of notices. Add/Edit/Delete/Toggle Pin. Pinned rows show 📌 badge.
